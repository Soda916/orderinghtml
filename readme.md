# 外帶點餐系統

簡單的前端頁面，提供外帶點餐、計算金額與複製訂單內容的功能。

## 菜單

- 鬆餅：原味 40、蜂蜜 45、草莓 45、巧克力 50（加棉花糖 +5）
- 陶喆餅乾：一片 25、兩片 40
- 飲料：雪碧／可樂 35、紅茶／綠茶 30、泰奶 75

## 功能
- 以表格式呈現完整菜單供對照，並提供快速點餐按鈕。
- 點擊菜單按鈕即可新增品項，重複點擊會增加數量。
- 可調整品項數量或使用紅色 `✕` 按鈕刪除品項。
- 輸入外帶單號（行動裝置會顯示數字鍵盤）。
- 切換顧客是否自備環保餐具，若自備將折抵 5 元。
- 自動計算應收金額並可一鍵複製訂單摘要（不含金額，複製後會清空訂單）。

## 使用方式
直接開啟 `index.html` 即可在瀏覽器中使用。
