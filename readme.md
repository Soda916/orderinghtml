# 外帶點餐系統

一個單頁前端工具，協助快速記錄外帶訂單、計算應收金額，並可複製訂單明細給內場或聊天室。

## 菜單

- 鬆餅：原味 40、蜂蜜 45、草莓 45、巧克力 50（加棉花糖 +5）
- 陶喆餅乾：一片 25、兩片 40
- 飲料：雪碧／可樂 35、紅茶／綠茶 30、泰奶 75

## 主要功能

- 首排顯示完整菜單版面，避免遺漏或誤記品項。
- 快速按鈕支援所有品項，鬆餅可一鍵加上「+棉」。
- 表格列出訂單內容，重複點擊按鈕會累加數量，可按紅色 `✕` 移除品項。
- 外帶單號欄位使用 `inputmode="numeric"`，手機會直接呼叫數字鍵盤。
- 切換顧客是否自備環保餐具，若自備會自動折抵 5 元。
- 即時計算小計／折扣與應收金額，並可複製不含金額的訂單摘要；複製後訂單會自動清空。

## 使用方式

以瀏覽器開啟 `index.html` 即可使用，無需額外安裝。

如需保留此次介面版本，可參考 `backups/index-original.html` 中備份的原始程式碼。
